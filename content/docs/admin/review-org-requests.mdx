## Organization Request Review

When users request to create a new organization, it goes into a review queue visible only to Admins.

### Reviewing Requests

1. Navigate to the **Organizations** page
2. Click the **Org Requests** tab (visible only to Admins)
3. You'll see all pending, approved, and rejected requests

### Approving a Request

1. Review the organization name and description
2. *(Optional)* Add **review notes**
3. Click **Approve**

The organization will be created and the requester will become its **Owner**. They'll receive an email notification.

### Rejecting a Request

1. *(Optional)* Add **review notes** explaining the reason
2. Click **Reject**

The requester will be notified by email with your notes.
