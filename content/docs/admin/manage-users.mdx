## User Management

Admins have access to a dedicated user management page at `/users`.

### Viewing Users

1. Click **Users** in the navigation bar at the top of the page (visible only to Admins)
2. A data table shows all registered users with columns for name, email, role, and subscription status
3. Use the column headers to **sort** or **filter** the list

### Changing a User's Role

1. Click on the **Role** cell for any user in the table
2. Select a new role from the dropdown: **Admin**, **Staff**, or blank (regular user)
3. The change saves automatically

### Editing User Details

1. **Double-click** any cell in the table to make it editable
2. Make your changes
3. Click the **Save** button that appears to confirm, or **Discard** to cancel

### Contacting a User

You can email any user directly from the table — see [Contact Members](/docs/organizations/contact-members#emailing-a-user-admin-only) for details.
