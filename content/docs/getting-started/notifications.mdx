## Push Notifications

A **bell icon** appears in the bottom-left corner of every page when you're signed in.

- Click it to **subscribe** to browser push notifications
- Click again to **unsubscribe**

When subscribed, you'll receive a browser notification whenever a new disaster request is created on the platform.

> **Note:** Your browser will ask for permission the first time. Push notifications require a modern browser that supports the Web Push API.

## Email Notifications

The platform sends automated emails for key events:

| Event | Who receives it |
|-------|----------------|
| Request created | The request creator (confirmation) |
| Someone responds to your request | The request owner |
| A response is cancelled | The request owner |
| Request is closed | The request owner + all responders |
| Organization approved/rejected | The requester |
| Join request submitted | Organization managers |
| Contact message sent | The farm owner or org owner |

Emails are sent automatically to the profile you signed up with. You can opt out of most email notifications by turning off **Email Notifications** in your profile settings (click **Edit Profile** from your profile page). Note that direct contact messages will still be delivered regardless of this setting.
