## How to Create a Request

1. Click the **New Request** button in the navigation bar at the top of the page
2. Select the **type of disaster** (fire, flood, hurricane, tornado, drought, frost, or other)
3. Select the **farm** affected by the disaster
5. *(Optional)* Add comments in the **description** with additional details about the situation
4. *(Optional)* Check **"Do you have a Gate you want to add?"** and select a gate from the dropdown — this tells responders which entrance to use
6. Click **Create Request**

After creating the request:
- You'll be redirected to the dashboard, centered on this request
- It will appear on the map as a marker
- All push notification subscribers will be notified
- You'll receive a confirmation email
